### 주요 기능

- 여러 사람에게 질문을 작성하고 공유할 수 있습니다.
- 질문에 대한 답변을 작성하고 공유할 수 있습니다.
- 👍좋아요👍 / 👎싫어요👎를 표시할 수 있습니다.
- 사용자의 눈을 보호하기 위한 다크모드를 지원합니다.
- 자신의 프로필을 쉽게 찾을 수 있도록 도와줍니다.
- 유효하지 않은 URL에 접속 시 404 페이지를 보여주며 홈으로 이동할 수 있도록 도와줍니다.
